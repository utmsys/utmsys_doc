3.1.1 Introduction

The WiFi LoRa 32 is a well-known IoT development board created by Heltec Automation. Since its release in 2017, it has been popular among developers and makers.

The new V3 version keeps the same pin layout as the V2 and continues to support Wi-Fi, Bluetooth Low Energy (BLE), LoRa, and an OLED display.

This board is ideal for applications in smart cities, smart farming, home automation, industrial control, security systems, wireless metering, and general IoT development.

Features​

  • Equipped with an ESP32-S3FN8 microprocessor, featuring a dual-core Xtensa® 32-bit LX7 CPU, five-stage pipeline, and clock speed up to 240 MHz.

  • Integrated SX1262 LoRa chip for long-range wireless communication.

  • USB Type-C interface with built-in voltage regulation, ESD protection, short-circuit protection, RF shielding, and other safety features.

  • Includes an SH1.25-2 battery connector and a built-in lithium battery management system that supports charging, discharging, overcharge protection, battery level monitoring, and automatic power switching between USB and battery.

  • Features a 0.96-inch OLED display (128×64 resolution) for showing debug info, battery status, and more.

  • Supports Wi-Fi, LoRa, and Bluetooth connectivity. Comes with a built-in 2.4GHz metal spring antenna for Wi-Fi and Bluetooth, and a reserved IPEX (U.FL) port for LoRa.

  • Uses the CP2102 USB-to-serial chip, making it easy to upload code and view debug output.

  • Compatible with the Arduino development environment for quick and flexible programming.

  • Designed with a high-quality RF circuit and low power consumption for efficient operation.
