ROS​

通过ros_gz 包进行 ROS 集成 。

支持的版本：

  • ROS 2 Jazzy（来自二进制）/Rolling（来自源代码）

对于 ROS 2 Rolling ，Rolling 分布将随着 REP-2000 中定义的 ROS 2 的下一个未来版本一起移动。
