Gazebo

提示

PC端安装

在Ubuntu上安装​

Harmonic 二进制文件适用于 Ubuntu Jammy (22.04) 和 Ubuntu Noble (24.04)。Harmonic 二进制文件托管在 packages.osrfoundation.org 仓库中。要安装所有 Harmonic 二进制文件， gz-harmonic 请安装相应的元包。

注意

对于 Gazebo Classic 如“gazebo11”用户：默认情况下，“gz-harmonic”无法与“gazebo11”同时安装。为了方便迁移，可以参考 “Gazebo11 与新版 Gazebo 并排安装”中的详细说明。

首先安装一些必要的工具：

sudo apt-get update
sudo apt-get install curl lsb-release gnupg

然后安装 Gazebo Harmonic：

sudo curl https://packages.osrfoundation.org/gazebo.gpg --output /usr/share/keyrings/pkgs-osrf-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/pkgs-osrf-archive-keyring.gpg] http://packages.osrfoundation.org/gazebo/ubuntu-stable $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/gazebo-stable.list > /dev/null
sudo apt-get update
sudo apt-get install gz-harmonic

所有库都应该可以使用并且 应用程序应该可以执行。 gz sim

返回入门 页面 开始使用 Gazebo！

卸载二进制安装​

如果您需要卸载 Gazebo 或在从二进制文件安装库后切换到基于源的安装，请运行以下命令：

sudo apt remove gz-harmonic && sudo apt autoremove
